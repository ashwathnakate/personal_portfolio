A modern, responsive portfolio website built with HTML, CSS, and JavaScript featuring glassmorphism design elements and smooth animations.

## Features

- **Fully Responsive Design** - Optimized for desktop, tablet, and mobile devices
- **Modern UI/UX** - Clean interface with glassmorphism effects and smooth transitions
- **Mobile Navigation** - Collapsible hamburger menu for mobile devices
- **Scroll Animations** - Interactive elements that animate on scroll
- **Sections Include:**
  - Hero section with social links
  - Statistics showcase
  - Technical skills overview
  - About section
  - Projects gallery
  - Publications & blogs
  - Contact information

## Technologies Used

- HTML5
- CSS3 (Grid, Flexbox, Custom Properties)
- Vanilla JavaScript
- Font Awesome Icons
- CSS Animations & Transitions

## Design Highlights

- Dark theme with cyan accent colors
- Grid background pattern
- Glassmorphism navigation bar
- Responsive grid layouts
- Smooth scrolling navigation
- Hover effects and micro-interactions
